# Activity Logs Firebase Schema

This document outlines the proposed schema for the `activity_logs` table in Firebase.

## Collection Name

`activity_logs`

## Document Schema

| Field         | Type      | Description                                                                                      |
|---------------|-----------|--------------------------------------------------------------------------------------------------|
| `logId`       | `string`  | A unique identifier for the log entry. This can be auto-generated by Firebase.                   |
| `timestamp`   | `timestamp` | The date and time when the activity occurred.                                                    |
| `userId`      | `string`  | The unique identifier of the user who performed the action.                                      |
| `userEmail`   | `string`  | The email address of the user who performed the action.                                          |
| `action`      | `string`  | The type of action performed (e.g., "generate_course", "complete_module", "delete_course").      |
| `details`     | `map`     | An object containing additional details about the action (e.g., the course ID, module ID).       |
